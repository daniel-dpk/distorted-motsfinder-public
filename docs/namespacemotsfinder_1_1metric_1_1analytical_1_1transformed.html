Create transformations suitable for TransformedMetric.

The transformations here modify the x and/or z coordinate such that MOTSs appear highly distorted even in case of a Schwarzschild slice. The purpose is purely to test the performance and accuracy of the MOTS finder and the quantities can be computed.

One advantage of using these presets over ad-hoc defined functions (i.e. not in a module) is that the resulting MOTS curves can be "pickled", i.e. stored to disk.
