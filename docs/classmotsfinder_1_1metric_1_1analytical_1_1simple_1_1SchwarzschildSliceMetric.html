Detailed Description

Spatial slice of Schwarzschild metric in isotropic coordinates.

In isotropic coordinates, the Schwarzschild spacetime metric becomes

\[ {}^{(4)}g = - \left(\frac{1-m/(2r)}{1+m/(2r)}\right)^2 dt^2 + \psi^4 \delta , \]

where \( \delta \) is the flat 3-metric and the conformal factor is

\[ \psi = 1 + \frac{m}{2r}. \]

See e.g. Equation (10.89) in [1] or Equation (3.16) in [2].

For a spatial slice, we just take the spatial part of this metric for constant \( t \).

Notes

Note that this is not the form the metric is usually written and as such, the physical quantities expressed in values of the "radial" parameter have different expressions. Specifically, the value of this parameter at which the horizon lies is given by (cf. [1]) \( r = m/2 \). This is not to be confused with the Schwarzschild radius \( \tilde r_s = 2m \), which has a geometric meaning related to the surface area of the horizon and at which the horizon lies in the traditional Schwarzschild coordinates.

Derivatives of the conformal factor are computed in psi() using

\[ \partial_i\psi = - \frac{m x^i}{2r^3}, \qquad \partial_i\partial_j\psi = - \frac{m \delta_{ij}}{2r^3} + \frac{3}{2} \frac{m x^i x^j}{r^5}. \]

References

[1] Schutz, Bernard. A first course in general relativity. Cambridge university press, 2009.

[2] Straumann, Norbert. General relativity. Springer Science & Business Media, 2004.

◆ horizon_area()

def horizon_area (   self)

Area of the Schwarzschild horizon.

The area is given by \(4 \pi \tilde r_s^2\), where \(\tilde r_s = 2m\) is the Schwarzschild radius. Note that in isotropic coordinates, the value of this radius cannot be taken as being a meaningful value of our r coordinate.

◆ horizon_coord_radius()

def horizon_coord_radius (   self)

Schwarzschild radius w.r.t. isotropic coordinates.

In the typical Schwarzschild coordinates, the horizon is at a "radial" coordinate 2m. This class implements the slice in isotropic coordinates, where the "radius" parameter value at which the horizon is located is m/2 instead.

Compute the value of psi at a given point.

Parameters
point3D point at which to evaluate psi.
derivativesWhether to compute the derivatives in addition to the value of psi itself. Default is False, i.e. compute only psi itself.
max_derivMaximum derivative order to return.
Returns
If derivatives==False, returns just the value psi(point). Otherwise, returns a tuple of derivatives of psi, starting at 0 up to max_deriv (inclusive). Each derivative of order > 0 is a nested sequence with indices i1, i2, ... meaning \(\partial_{i_1}\partial_{i_2}\psi\).
