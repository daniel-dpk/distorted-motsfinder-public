def motsfinder.exprs.common._update_domains (   obj)
private

(Re-)Generate domainX/Y/Z attributes from the current domain.

This is a convenience for higher (up to 3) dimensional functions where a stored domain of the form (e.g.) 

obj.domain = ((a1, b1), (a2, b2))

is used to create/update the attributes 

obj.domainX = (a1, b1)
obj.domainY = (a2, b2)

def motsfinder.exprs.common.is_zero_function (   func)

Check whether a given function is the zero function.

This checks the identity of the given function with a particular zero function. This can be useful if expressions/evaluators actually use/return this _zero_function() function object when they know this is correct.
