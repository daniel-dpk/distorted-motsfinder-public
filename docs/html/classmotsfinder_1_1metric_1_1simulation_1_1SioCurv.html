Detailed Description

Extrinsic curvature of a slice read from simulation data.

Objects of this type are usually created from SioMetric objects to represent the extrinsic curvature of the slice. This class uses the configuration of the respective metric object and hence stores a reference to it as instance variable. The metric's data is not actually used, just its configuration (like the interpolation settings or the data file name).

Note that the metric object is responsible for resetting this object's cache when evaluation is requested at a different point than before. Hence, this object informs the metric where it is being evaluated.

Evaluate/interpolate the extrinsic curvature at a point.

This respects the interpolate attribute of the corresponding metric object.

Returns
If diff=0, returns a 3x3 matrix representing the extrinsic curvature components at the given point. If diff=1, returns a 3x3x3 matrix dk, where dK[i,j,k] corresponds to .
Parameters
pointPoint in coordinate space at which to evaluate/interpolate.
diffDerivative order to compute. Default is 0.

Notes

Often the curvature and its derivative are both needed for computations at a particular point. This class hence stores the curvature tensor at the last point it was computed making consecutive calls with diff=0 and diff=1 more efficient.

def reset_cache (   self)

Reset cached values to force re-evaluation upon next request.

This should be called when the point at which to evaluate/interpolate is different from the last point at which evaluation was done. Users usually don't need to call this themselves as it is called indirectly by the updating mechanism.
