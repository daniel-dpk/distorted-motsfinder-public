Detailed Description

Base class for surfaces with S^2 topology.

The surface objects are callable classes mapping (theta, phi) to (x, y, z), where 0 < theta < pi and 0 <= phi < 2 pi.

The conventions for parameter names are as follows:

  • ta: angle theta
  • ph: angle phi
  • uv: coordinates on S^2, e.g. uv=[ta, ph]
  • mu: derivative order w.r.t. ta
  • nu: derivative order w.r.t. ph

Base class init.

Parameters
metricThe Riemannian metric of the space this surface is immersed in. Default is to use the flat Riemannian 3-metric.
scale_normalWhether to scale the normal vector by 1/sin(ta). This creates more uniform normal lengths across the surface, specifically near the poles. Default is True.
nameA name which may be used for printing or plot labels.
fd_frechet_derivativesWhether to use analytical (False, default) or finite difference (True) methods to compute the partial Frechet derivative of the expansion w.r.t. derivative orders of the horizon function. This is mainly a debugging option used to verify the implemented analytical formulas.
