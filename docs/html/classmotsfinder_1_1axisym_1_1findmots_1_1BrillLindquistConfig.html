Detailed Description

Configuration for finding MOTSs in Brill-Lindquist metrics.

This is a specialization of the MotsFindingConfig for Brill-Lindquist configurations. By specifying a distance parameter d and the two masses, the metric can be created dynamically.

See the documentation of MotsFindingConfig for a detailed description of all available options.

Create a Brill-Lindquist find_mots() configuration object.

This shares all the parameters of MotsFindingConfig (see there for the documentation). The additional parameters are described next.

Parameters
dDistance parameter for the Brill-Lindquist metric.
m1Mass parameter (i.e. not physical/irreducible mass) of the black hole with puncture on the positive z-axis.
m2As m1 but for the puncture on the negative z-axis.
