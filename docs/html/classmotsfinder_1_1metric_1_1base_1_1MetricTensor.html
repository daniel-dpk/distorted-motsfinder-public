Detailed Description

Metric tensor class at a given point.

The metric tensor is represented as a matrix, which means that some basis has been chosen. Operations with vectors are only well-defined if they are given in components w.r.t. the same basis.

There are convenience methods for raising and lowering indices of covectors and vectors, respectively, and calling this object with two vectors will compute the scalar product.

The inverse matrix (representing the metric with "indices up") is computed on-demand and then cached for subsequent use.
