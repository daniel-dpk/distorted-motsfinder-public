Detailed Description

Tweaked baseclass for unit tests.

By deriving from this class, you:

  • Get timed individual tests (needs verbosity=2) if TestSettings.timing is true.
  • Can implement a hook (failureHook()) that is called after a test has failed (or errored). It is called before tearDown(), allowing you to, for example, collect generated files for inspection before they are deleted.
