Integrate a function using fixed order Gaussian quadrature.

This is a wrapper for scipy.integrate.fixed_quad(). Given a full domain to integrate, it uses the information of the current sub-domain to choose less than the full set of n points such that integrating the full domain in several intervals, the total number of evaluations is approximately n.

Parameters
funcFunction taking a list of values and returning a list of results.
a,bInterval to integrate over.
nOrder of the Gaussian quadrature for the full_domain.
full_domainFull domain to eventually integrate over. If not given, a,b is taken to be the full domain, i.e. no reduction of quadrature order is done.
min_nMinimum quadrature order for very small sub-domains.

def motsfinder.axisym.curve.expcurve.alm_matrix (   alm)

Convert a sequence of a_lm coefficients into matrix form.

The alm sequence returned by e.g. ExpansionCurve.expand_shear_scalar() (when compress=False is set) is a flattened (1D) sequence of values that represent all the values , where l = 0, ..., l_max and m = -l, ..., +l. This function takes the flattened sequence alm and returns the complex valued matrix of a[l,m] coefficients.

Note that this is a pure Python implementation not suitable for large matrices.

Evaluate the expanded function at a set of points.

Given a list of coefficients a_l0 of a function expanded into spin weighted spherical harmonics, synthesize the function on a grid of num points.

Returns
xs, ys, where xs is an array of theta values at which the function has been evaluated and ys the corresponding function values.
Parameters
al0Coefficients a_l0, where l = 0, ..., lmax.
numNumber of theta-values at which to evaluate the function. The default is to use the points that can be used to expand the function into exactly the same al0 coefficients.
spinweightThe spin weight of the functions expanded into. Default is 2.
realWhether to drop any imaginary part and return an array of real values. Default is True, which is suitable if the expanded function had vanishing imaginary part in the first place.
