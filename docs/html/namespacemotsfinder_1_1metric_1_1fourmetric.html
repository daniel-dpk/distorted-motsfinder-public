Detailed Description

Spacetime metric constructed from 3-metric and lapse, shift.

This represents a spacetime 4-metric which is constructed from the spatial slice 3-metric and lapse/shift fields. To compute derivatives of this metric (in order to construct e.g. the Christoffel symbols), the extrinsic curvature needs to be supplied as well.
