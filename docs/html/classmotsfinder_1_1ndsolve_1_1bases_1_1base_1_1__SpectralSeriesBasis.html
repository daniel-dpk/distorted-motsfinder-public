Detailed Description

Spectral basis base class utilizing implemented series expansions.

This convenience base class simplifies implementation of spectral bases for which a series expression has been implemented as a subclass of exprs.series.SeriesExpression.

Child classes need only provide the methods:

  • get_series_cls()
  • evaluate_all_at()
  • _compute_deriv_mat()

Create a basis object with a given domain and resolution.

Parameters
domain(2-tuple/list) Physical domain as (a, b) of the problem. This is independent of the domain of the basis functions.
num(int) Number of basis functions to expand the solution in. This defines the resolution (and computational cost).
lobatto(boolean, optional) Whether to use the Gauss-Lobatto points (default), which include the two endpoints, or the Gauss points, which don't.
