Plot a surface parameterized as z = f(x, y).

Parameters not described here are described in plot_scatter() or plot_1d().

Parameters
fFunction to plot. Should take a 2-tuple/list/array and return a floating type. Alternatively, this may be a (NumPy) matrix which will be plotted as f(i, j) = mat[i, j], where i and j run over the full shape (i.e. ranges and points arguments are ignored).
domainDomain to plot f on as ((x1, x2), (y1, y2)). If not given, the function's domainX and domainY attributes are queried. If these don't exist, the interval (0,1) is used.
rel_rangeSimilar as rel_range in plot_1d(), but containing the relative range to plot for both axes.
pointsNumber of samples to take in each direction. The function will be evaluated on a grid of points*points points. May also be a 2-tuple/list with a resolution for each axis. Default is 30.
lwLine width for drawing the edges of the 3D mesh. Default is 0.4.
cColor of the mesh surfaces. Default is white, i.e. 'w'.
ecEdge color. Default is black, i.e. 'k'.
cmapOptional color map to indicate height of the surface.
absoluteIf True, uses the absolute of the values returned by f.
zoffsetValue to add to the values returned by f.
zlogWhether to plot the base-10 logarithm of the function values.

Create a scatter plot of 3D points.

Options are similar to the ones of plot_1d() with the below additions.

Parameters
pointsIterable of 3D points to plot.
sSize of the individual point markers. Default is 20.
cColor of the markers. Default is blue, i.e. 'b'.
zlimOptional interval of the z-axis to show. Default is automatic.
zscilimitsAs the xscilimits parameter but for the z-axis.
ztick_spacingAs xtick_spacing but for the z-axis.
zlabelLabel for the z-axis
azimAzimuthal angle used for the 3D projection. Default is -60.
elevElevation angle used for the 3D projection. Default is 32.
distCustom "distance" to the plotted area. Modifies perspective and can help in cases where important parts are cropped off the image.

Create a 3D surface plot with manual triangulation.

Options not described here are similar to the ones of plot_2d().

Parameters
pointsVertex points of the mesh
trianglesList of 3-tuples indicating which vertices form triangles. If not given, points will be interpreted as an object with a surface() method returning the actual points and triangles.
lw(float, optional) Linewidth of the triangle edges. Default is .4.
c(color, optional) Color of the triangle faces. May be overridden by facecolors.
ec(color, optional) Edge color for the triangle edges. Set to None to not draw the wire.
facecolors(iterable of floats, optional) A value assigned to each "face" (i.e. triangle). The values will be associated with colors of the colormap specified via cmap.
cmap(colormap, optional) A colormap (e.g. one of the matplotlib.cm properties) to use for the face colors. Has no effect without facecolors.
colorbar(boolean, optional) Whether to show a color bar indicating the values associated with colors of the cmap. Has no effect without facecolors.
