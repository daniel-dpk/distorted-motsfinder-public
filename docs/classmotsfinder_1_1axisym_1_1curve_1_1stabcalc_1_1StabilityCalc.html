Detailed Description

Manage computation of quantities to evaluate stability operator.

Given a MOTS and a point on the MOTS, this class computes the various quantities needed to evaluate the MOTS stability operator [1]. Notation is mostly compatible with [1]. In particular, we use \(k_\mu \ell^\mu = -2\) unless otherwise noted in the docstrings.

Note that we assume vacuum throughout.

References

[1] Andersson, Lars, Marc Mars, and Walter Simon. "Stability of marginally outer trapped surfaces and existence of marginally outer trapped tubes." arXiv preprint arXiv:0704.2889 (2007).

◆ compute_shear()

def compute_shear (   self)

Compute the shear tensor in coordinates on the surface.

This computes \(\sigma_{AB}\), where

\[ \sqrt{2} \sigma_{AB} = \nabla_A \ell_B - \frac12 q_{AB} q^{CD} \nabla_C \ell_D = -K_{AB}^\mu\ell_\mu + \frac12 q_{AB} q^{CD} K_{CD}^\mu \ell_\mu \,. \]

The factor of \(\sqrt{2}\) results from our definition of the ingoing and outgoing null normals \(k^\mu\) and \(\ell^\mu\), i.e. we use a scaling where \(\ell^\mu k_\mu = -2\), whereas e.g. [1] uses \(\ell^\mu k_\mu = -1\). This means the values computed here are compatible with [1].

Also note that for a MOTS, the second term above vanishes by definition.

References

[1] Dreyer, O., Krishnan, B., Shoemaker, D., & Schnetter, E. (2003). Introduction to isolated horizons in numerical relativity. Physical Review D, 67(2), 024018.

def compute_shear_k (   self,

Compute the shear of the ingoing null normal.

Similar to compute_shear(), this computes the symmetric tensor w.r.t. coordinates on the surface, i.e.

\[ \sigma_{AB} = \frac{1}{\sqrt2} (\nabla_A k_B - \frac12 q_{AB} q^{CD} \nabla_C k_D) = \frac{1}{\sqrt2} (-K_{AB}^\mu k_\mu - \frac12 q_{AB} \Theta_{(k)}) \,. \]

The factor of \(1/\sqrt2\) makes this result compatible with the convention \(\ell \cdot k = -1\).

Parameters
full_outputIf False (default), return just the shear of k. If True, return
  • the symmetric tracefree shear of k as 2x2 matrix w.r.t. the current coordinates on the MOTS
  • the (coordinate independent) square of this shear
  • the expansion of k
  • the trace of the shear (for diagnostic/debugging purposes)

def surface_gravity (   self,

  X = None

Compute a (slicing dependent) surface gravity.

See .expcurve.ExpansionCurve.surface_gravity() for more information. Computes

\[ \kappa^{(X)} = - k_\mu X^\nu \nabla_\nu \ell^\mu \,, \]

where \(X^\mu = \ell^\mu\) by default.

Uses the convention \(\ell \cdot k = -1\), in particular

\[ \ell^\mu = \frac{1}{\sqrt{2}} (n^\mu + v^\mu) \,, \qquad k^\mu = \frac{1}{\sqrt{2}} (n^\mu - v^\mu) \,. \]

Parameters
dt_normalThe term \(\partial_t v^\mu\), where v is the normal of the MOTS within the slice. Note that this should not simply be the time derivative of the normal's components. See .expcurve.ExpansionCurve.dt_normals().
X4-vector with respect to wich to compute the surface gravity. Default is to use the outgoing null normal \(\ell^\mu\).
