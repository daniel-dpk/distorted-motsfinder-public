Detailed Description

Python MOTS finder project.

Currently, the only implemented finders are located in the motsfinder.axisym package and assume axisymmetry. The horizon \( \sigma \) is parameterized either in the traditional way as \( r = h(\theta) \) or via coordinates w.r.t. a reference surface \( \sigma_R \).

To solve the resulting elliptic ODE for the horizon function, a pseudospectral solver in the motsfinder.ndsolve module is used.

The analytically implemented metrics in the motsfinder.metric module can serve as testing cases.
