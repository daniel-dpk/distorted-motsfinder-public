Detailed Description

Expansion cache/calculator for StarShapedCurve objects.

This computes and caches interim results of the expansion and its functional derivatives w.r.t. the horizon function for curves in star-shaped parameterization.

This object is usually only created by the StarShapedCurve object.

def _compute_s_ds_X_Y (   self)
private

Compute the terms we need to compute the expansion.

To compute the covariant normal \( s_i \) on the curve represented by the StarShapedCurve, we define the function

\[ F(\vec x) = r - h(\lambda(\vec x)), \]

This means that on the surface defined by h we will have F = 0. The covariant normal is therefore obtained via

\[ s_i = \nabla_i F = \partial_i F = \frac{x^i}{r} - X_i h', \]

where \( X_i = \partial_i\lambda \), compare Thornburg [1]. We furthermore need the derivatives

\[ \partial_i s_j = \frac{\delta_{ij}}{r} - \frac{x^i x^j}{r^3} - X_{ij}h' - X_i X_j h'', \]

where \( \delta_{ij} \) is the Kronecker delta and we set \( X_{ij} = \partial_i\partial_j\lambda \). Since this will always be evaluated on the curve (trial surface), we replace \( r \to h \) in the following.

Note that since we cannot differentiate between variables based on how many indices they have, we use Y for the components \( X_{ij} \).

Since we know the coordinate transformations analytically,

\[ \cos\lambda = \frac{z}{r}, \]

we can easily compute all the required terms and obtain

\[ X_1 = \frac{\cos\lambda}{h}, \qquad X_2 = 0, \qquad X_3 = -\frac{\sin\lambda}{h}, \]

\[ X_{11} = -\frac{\sin(2\lambda)}{h^2}, \qquad X_{22} = \frac{\cot\lambda}{h^2}, \qquad X_{33} = \frac{\sin(2\lambda)}{h^2}, \qquad X_{13} = -\frac{\cos(2\lambda)}{h^2} = X_{31}, \]

\[ s_1 = \sin\lambda-\frac{h'}{h} \cos\lambda, \qquad s_2 = 0, \qquad s_3 = \cos\lambda+\frac{h'}{h} \sin\lambda, \]

\begin{eqnarray*} \partial_1 s_1 &=& \frac{1}{h^2} \left( (h-h'')\cos^2\lambda + h'\sin(2\lambda) \right), \\ \partial_2 s_2 &=& \frac{1}{h^2} \left( h-h' \frac{\cos\lambda}{\sin\lambda} \right), \\ \partial_3 s_3 &=& \frac{1}{h^2} \left( (h-h'')\sin^2\lambda - h'\sin(2\lambda) \right), \\ \partial_1 s_3 &=& \frac{1}{2h^2} \left( (h''-h)\sin(2\lambda) + 2h'\cos(2\lambda) \right) = \partial_3 s_1. \end{eqnarray*}
