Detailed Description

Base class for trigonometric series expressions.

This unifies all common aspects of truncated sine and cosine series expressions.

Parameters

nameName of the expression (e.g. for print_tree()).

Create all frequencies for the used basis.

A cosine series consists of all cos-frequencies, starting from 0, while for a sine series, we should start from 1.
